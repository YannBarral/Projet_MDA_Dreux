Press Coverage Options

The Press Coverage tab contains options to modify the behavior of the press coverage list.

URL

This is the permalink of the press coverage page. Take care when changing this option as it will affect page and feed links. Your cache may need to be cleared when changing this option.

Featured Image

This is the default featured image that will appear if one is not added for a press coverage.

Link Target

By default press coverage links will open in a new browser window when clicked (e.g. _blank). Removing this value will disable this option.
