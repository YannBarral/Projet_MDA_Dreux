Shortcode Options

These options modify the default behavior of the nooz shortcode. It is recommended that you change these options to fit the majority of your use-cases when using the nooz shortcode. For detailed adjustments, you can modify each option on a per-shortcode basis.

  • Display Count

    Use this option to set the number of press releases or coverage to display per shortcode/page.

  • Display Type

    This option modifies the view output. Selecting group will list items grouped by year and list will display a simple list. Press releases and coverage are displayed in reverse chronological order.

  • Date Format

    Dates are displayed along with each list item. This option allows you to change the date format of the displayed dates. Leave this blank to use the default date format as set in WordPress. Learn more about formatting dates.

  • Display Excerpts

    This option allows you to turn on/off the display of excerpts. An excerpt will only be displayed if available for the specific press release or coverage.

  • Read More Links

    The read more link, by default will follow the excerpt, providing the reader a link to the full press release or coverage.

    When read more is enabled, you will be able to change the read more link text.

  • Pagination Links

    When you have many press releases or coverage, it is recommended that you break them up into pages. Use the Display Count option to set the number of press releases or coverage per page.

    When pagination is enabled, you will be able to change the previous and next link text. These are the links that the reader will use to navigate forward and backwards through all the press releases and coverage items.
